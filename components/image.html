Image

Images can be made responsive with .img-fluid class that is applied to the image so that it scales with the parent element.

Responsive

The default form of an image.

<img src="../assets/images/flower1.jpg" class="img-fluid" />

Circle Image

Circular form of an image.

<img src="..." class="img-fluid circle" />

Object Image

The default form of an image. To use in small objects like avatars.

<img src="..." class="circle object" />
