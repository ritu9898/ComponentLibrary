Input

An input field that is used to take inputs like username, password, or many more. Usually used in form tags.

Default

The default form of an input box.

<div class="input-group"> <input class="" placeholder="Username" /> </div>

Sizing

Set heights using classes like .input-lg and .input-sm.

<div class="input-group"> <input class="input-lg" placeholder="Large" /> </div> <div class="input-group"> <input class="" placeholder="Default" /> </div> <div class="input-group"> <input class="input-sm" placeholder="Small" /> </div>

Disabled

Add the disabled boolean attribute on an input to prevent user interactions and make it appear lighter.

<div class="input-group"> <input class="input-disabled" placeholder="Username" disabled /> </div>

Validations

We recommend using client side validation, but in case you require server side, you can indicate invalid and valid form fields with .is-invalid and .is-valid. Note that .invalid-feedback is also supported with these classes.

Please provide some input
<div class="input-group"> <input class="invalid-input" placeholder="Username" /> <div class="invalid-input-msg"> Please provide some input </div> </div>
